# FDert - Free Resource Hub

A comprehensive collection of free tools, software, and resources organized by category.

## Features
- Over 8,000 links across various categories.
- Elegant dark/light mode interface.
- Instant search and keyboard shortcuts (Ctrl+K).
- Hash-based routing for direct link sharing.
- Serverless feedback system.

## Data Structure
The data is stored in `data.json`, which is consumed by `index.html`. 

## Deployment
This project is ready for deployment on Vercel.

### Feedback System Setup
To enable the feedback system:
1. Create a Discord Webhook (or any JSON-accepting webhook).
2. Set the `WEBHOOK_URL` environment variable in your Vercel project settings to your webhook URL.
